# search-engine

Ubtuntu 20.04 installation:

```bash
sudo apt update
sudo apt install python3-pip
sudo apt install gunicorn
sudo ufw enable
sudo ufw allow 8080

bash install.sh
```

Run:

```bash
bash start.sh
```
